# 🚀 Towers Ing
Portal educativo para la identificación de roles colaborativos en equipos técnicos.

## 📁 Estructura
- `index.html` → Página principal
- `estudiante.html` → Formulario de identificación de rol
- `administrador.html` → Gestión de registros y grupos
- `dashboard/` → Estadísticas y control
- `blog/` → Artículos técnicos
- `cv/` → Currículum profesional
- `proyectos/` → Desarrollos tecnológicos
- `forms/` → Herramientas interactivas
- `training/` → Cursos y talleres

## 🛠️ Tecnologías
HTML5, CSS3, JavaScript, localStorage.

## 🌐 Despliegue
Listo para desplegar en Vercel, Netlify o GitHub Pages.
